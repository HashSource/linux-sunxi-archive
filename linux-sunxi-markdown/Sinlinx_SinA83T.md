# Identification
The PCB has the following silkscreened on it: 
[code] 
    SINA83T 芯灵思 | SINLINX
[/code]
In android, under Settings->About Tablet, you will find: 
  * Model Number: _SINLINX A83_
  * Build Number: _octopus_f1-eng 5.1.1 LMY48B 20161225 test-keys_
