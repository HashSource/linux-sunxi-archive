Identification

Find out the strings as reported under settings.

In android, under Settings->About Tablet, you will find:

  • Model Number: DEVICE
  • Build Number: SOC_BOARD_DEVICE_*.*

The PCB shows: 

TZX-Q8-713B6

Manual build

  • For building u-boot, use the "TZX-Q8-713B6" target.
  • The .fex file can be found in sunxi-boards as tzx-q8-713b6.fex

Everything else is the same as the manual build howto.

Locating the UART

Uart has 2 pads on the back of the PCB (near the SoC). On A13 the UART's is on pins 151 and 152, which are left of the SoC. Just solder on some wires according to the UART howto
