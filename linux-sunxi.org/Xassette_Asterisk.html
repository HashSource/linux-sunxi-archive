Xassette-Asterisk is an open-source reference design for D1s developed by SdtElectronics. It was released in November 2021 and exposed all peripherals of D1s via a range of connectors, in which USB, headphones, line-in, RGB LCD, DVP camera, SD and SPI flash have on-board standard interface. The hardware design is open-source under the CERN OHL-w v2 licence.

Identification

The mark "Xassette-Asterisk" can be found at the back of the PCB.

Sunxi support

Current status

As D1s has little difference between D1, the tina BSP for D1 can be adopted to D1s without much modification.

Images

An image ready to be burned into the card is at the release page in the Github repository

Tips, Tricks, Caveats

FEL mode

The FEL button triggers FEL mode. The xfel tool has support for the D1s chip.
