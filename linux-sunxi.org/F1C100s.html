Website Product Page


The Allwinner F1C100s is a small SoC with an ARM926EJ-S CPU core and 32MB of embedded DDR SDRAM.
The Allwinner F1C200s is a small SoC with an ARM926EJ-S CPU core and 64MB of embedded DDR SDRAM.

The Allwinner F1C500s and F1C600s are now EOL.

Overview

Main components of the F1C100s & F1C200s

  • CPU: ARM926EJ-S (ARMv5TE) @ 533MHz
    • L1 Cache: 16KB Data, 32KB Instruction
    • FPU: None
  • GPU: Custom 2D-only
  • VPU: Unknown
