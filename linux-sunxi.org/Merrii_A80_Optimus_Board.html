The A80 Optimus Board is a official development board for the Allwinner A80 SoC, targeting both developers and consumers.

Identification

The board has a clearly recognizable Allwinner logo, the name "Allwinner Tech", and "OPTIMUS BOARD", all uppercase, in the lower right hand corner.

Sunxi support

Current status

Sunxi Kernel

Like all A80 based devices, there is no support in our sunxi kernel.

Sunxi U-Boot

Like all A80 based devices, there is no support in u-boot-sunxi.

Mainline kernel

Mainline kernel support is queued up for 3.19. Currently only UARTs are supported.

Use the sun9i-a80-optimus.dtb device-tree file for the mainline kernel.

Mainline u-boot

Like all A80 based devices, there is no support in mainline u-boot.

Tips, Tricks, Caveats

FEL mode

There is no FEL mode button on this board.

Instead you can enter fel mode from U-Boot. While U-Boot is starting, type CTRL+C to interrupt it. At the prompt type efex to enter fel mode.

If booting fails entirely then FEL mode can be entered by shorting pin 29-30 on the nand/emmc with a suitable metal object (tweezer or the like), by vendor official procedure.
